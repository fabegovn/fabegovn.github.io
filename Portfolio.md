# DỰ ÁN NỔI BẬT

## 1. Command Logic

### Thể loại

Tactical Puzzle Game

### Mô tả

Command Logic là một trò chơi giải đố chiến thuật trên lưới, nơi người chơi điều khiển ba robot với các kỹ năng khác nhau để hoàn thành nhiệm vụ trong số lượt giới hạn.

Người chơi phải:

* Lập kế hoạch di chuyển
* Điều phối robot
* Vượt qua drone tuần tra
* Tương tác với cửa, công tắc và vật cản

### Điểm nổi bật

* Multi-agent puzzle solving
* Turn-based tactical gameplay
* AI Mission Generator
* Dynamic level creation bằng AI

### Công nghệ

* HTML5
* CSS3
* JavaScript
* OpenAI API

### Vai trò

* Thiết kế gameplay
* Thiết kế level system
* Xây dựng game logic
* Tích hợp AI tạo màn chơi

## 2. Mật Hình

### Thể loại

Word Puzzle Game

### Mô tả

Mật Hình là trò chơi đố hình đoán chữ tiếng Việt.

Người chơi quan sát các hình ảnh, suy luận ý nghĩa và ghép chúng lại thành từ hoặc cụm từ chính xác.

### Điểm nổi bật

* Puzzle dựa trên ngôn ngữ tiếng Việt
* Hệ thống gợi ý AI
* Hình nền tạo bởi AI
* Hệ thống xoay vòng câu đố ngẫu nhiên

### Công nghệ

* HTML5
* CSS3
* JavaScript
* Pollinations AI

### Vai trò

* Thiết kế gameplay
* Xây dựng hệ thống câu đố
* Tích hợp AI
* Thiết kế trải nghiệm người dùng
